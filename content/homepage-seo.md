**Meta description (155 chars):**
ISO, DIN, ASME & AS fastener standards explained for engineers and buyers — plus free interactive torque, grade, and unit conversion tools.

---

## Notes
- Title stays under 60 characters so it doesn't truncate in SERPs.
- Description leads with the standards bodies (high-intent keyword match for
  engineers/buyers searching "ISO fastener standard," "DIN bolt torque," etc.)
  and closes with the interactive-tools hook to lift CTR.
- Set both via Ghost Admin → Settings → General → "Meta data" (these map to
  `{{meta_title}}` / `{{meta_description}}` in `default.hbs`), or override
  per-post via each post's SEO settings.
